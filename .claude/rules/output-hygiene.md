# Гигиена вывода

У LLM конечное контекстное окно. Шумный вывод вытесняет полезную информацию.

- Не выводить большие структуры данных целиком → метаданные (размер, типы, статистика)
- Не печатать полные коллекции → агрегаты: top-N, перцентили, counts
- Verbose вывод → `logs/`, не в stdout
- После значимого действия: одна строка результата с датой и ключевыми числами

✓ `[ГГГГ-ММ-ДД] описание: ключевая метрика/итог (+дельта к точке отсчёта если применимо)`
✗ дамп всей структуры данных в чат на тысячи строк
