# Роли агентов

Для сложных задач запускать специализированных агентов с разными worktrees:

- **feature-engineer** — исследует и реализует одну группу фичей
- **model-tuner** — гиперпараметры на фиксированном feature set
- **data-validator** — проверка утечек, схемы, сдвигов распределений
- **results-documenter** — обновляет PROGRESS.md, пишет summary эксперимента
- **code-reviewer** — проверяет утечки, векторизацию, корректность сплитов

Назначать разные файлы/модули разным агентам — не давать двум агентам
трогать один файл одновременно.

Координатор обновляет PROGRESS.md после завершения каждого субагента.
