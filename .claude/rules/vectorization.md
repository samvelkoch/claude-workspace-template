# Векторизация — жёсткое правило

Только векторизованные операции. Никаких циклов по большим данным.

✓ pandas/numpy операции по колонкам и группам
✓ groupby + transform/agg
✓ vectorized string ops (.str.), boolean indexing
✗ for row in df.iterrows() — никогда
✗ apply() с Python-функцией на миллионах строк — только если нет альтернативы, объяснить почему

Если нужен цикл — сначала спросить: можно ли переписать через vectorized?
