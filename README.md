## Быстрый старт

```bash
git clone <repo>
cd <repo>
cp .env.example .env                    # заполнить креды (Confluence, БД, прочее)
echo "$(hostname -s)" > .claude/host-id # короткий id этой машины для Handoff
just install                            # установить зависимости
just baseline-quick                     # быстрая проверка что окружение работает
```

## Воспроизведение baseline

```bash
just baseline-full            # полный прогон, метрики записываются в logs/
```

Ожидаемые метрики — см. секцию `## Baseline` в `PROGRESS.md`.
Если не воспроизводится — стоп, см. `CLAUDE.md`.

## Структура

- `CLAUDE.md` — протокол работы для Claude (читается агентом каждую сессию)
- `PROGRESS.md` — операционный дневник: статус, baseline, лог результатов
- `context/` — знания о домене (схема данных, архитектурные решения)
- `data/sample/` — сэмплы; `data/raw/` и `data/full/` — не в git
- `src/` — код пайплайна; `notebooks/` — исследовательские ноутбуки
- `.claude/` — правила и команды для агента
- `justfile` — единая точка входа для частых команд (см. `just --list`)
