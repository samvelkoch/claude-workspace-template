# Архитектурные решения

Что выбрано в проекте и **почему**. Дополняется по мере принятия решений.
Зеркало секции `## Архитектурные решения` в `CLAUDE.md` — здесь подробности
и альтернативы, там короткие тезисы для активного контекста.

## Формат записи

```
## [ГГГГ-ММ-ДД] <Решение одной строкой>

**Контекст:** что заставило принять решение (какая задача / альтернативы / что не работало).
**Решение:** что выбрали.
**Причина:** почему именно это.
**Альтернативы:** что ещё рассматривали и почему отбросили.
**Последствия:** что теперь нельзя / приходится делать иначе из-за этого решения.
```
